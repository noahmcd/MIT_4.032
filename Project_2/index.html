Project 2: Weather Visualization

Created by Noah McDaniel

Visualization 1

best outdoor conditions today are at

This visualization displays the PMV index, which measures comfortability as a function of various weather effects. The intent is for users to be able to identify the best and work conditions of the day, so that they can plan their dress and activities accordingly

I chose this measurmeent scheme because I wanted to experiment with efficent data communication. "Comfort" is a way in which complex data, in this case weather, can be distilled into a simple number or word. While we lose some understanding of the details, the accessibility is enhanced.

Visualization 2

Cambridge, next 2 days

This visualization displays each of the subsequent 48 hours as a circle. The radius indicates precipitation, and the color indicates temperature. I decided to omit scales because the visualization is meant to depict weather artistically, and is not intended to be used as an analytic tool.

This design changed slightly from my plan, as I found the use of space in a matrix layout more visually appeaing. I think it would be fun to compare diagrams such as these between places around the world, to see how they are similar and different.
