4.032: Visualizing Time

Clock 1

This clock is a spirogram, which draws a circle every second and completes a full diagram every minute. The colors are generated from a hex code, converted from the time. Thus, displaying the numerical time is unnecessary if the viewer can decode the colors. This clock represents time as a gradient, flowing around space from color to color.

Clock 2

This clock is a strand of DNA. I was thinking about fundamental forms of data storage, and realized that DNA is one of the most essential forms of data in existence. There are 4 bases, which can be used to convert time into a base-4 system, which I then represent as letters in the strand. It takes 3 bases to write an hour, second, or minute. The time reads downward.

Clock 3

This is a concept in progress, but I wanted to experiment with gradients and a mor eminimalist design. I decided to forgo a tradional clock face or numbering because time has no absolute reference point. The sharp contrast between the hands and the background represents absoluteness versus infiniteness.
